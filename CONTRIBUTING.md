# Contributing to LibreThinker Extension

Thanks for taking the time to contribute!

As this extension is serving thousands of people, I've decided to set a few rules which are hopefully reasonable:

1. The update should not degrade the experience for other users; eg introduce potential bugs or cause confusion in the UI
2. Changes should be as small as possible; it's ok to submit multiple PRs for different features
3. While you can use AI to generate code, please check that everything makes sense in the code before submitting the PR

While not necessary, I also recommend [contacting me](https://tally.so/r/jaZx41) beforehand. 
I might be in the process of making major changes, and your PR may not be compatible with the updates.
